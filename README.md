# TohouFantasy
-A 2d shooting game  
-using C/C++  
## How to play
- open the planewar.exe in Release dir  
- w,a,s,d or ↑，↓, ←, → for move  
- z for shooting  
- x for ultimate (only when you have sprite around you)
